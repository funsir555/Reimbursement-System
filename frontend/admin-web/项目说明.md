## 当前结构

```text
frontend/admin-web/
├── src/
│   ├── api/               # API 兼容层与分域 modules
│   ├── layouts/           # 主布局与工作台骨架
│   ├── router/            # 路由注册与导航配置
│   ├── stores/            # Pinia 状态
│   ├── views/             # 各业务域页面
│   └── components/        # 可复用组件
```

## 当前前端职责

前端当前主要承载：

- 登录与登录态维持
- 首页概览与个人入口
- 报销创建、列表、审批、查询、详情
- 流程管理与设计器相关页面
- 财务、档案、固定资产、archive-agent 等工作台页面
- 权限控制、菜单导航、路由守卫

## 当前约束

- `src/api/index.ts` 只做兼容导出，不再回灌新的领域真相
- 分域 API、类型和 request wrapper 优先进入 `src/api/modules/`
- 前端专题说明与当前进度以主治理文档为准，不在本文件里重复维护另一套优先级

## 常用命令

```bash
cd frontend/admin-web
npm install
npm run dev
npm run test:unit
npm run build
```
